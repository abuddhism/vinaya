### **Overview**

This repository contains the complete research output of a large-scale study on the evolution of Buddhist Vinaya rules across three traditions (Theravāda, Dharmaguptaka, Mūlasarvāstivāda). The main findings and data are presented as PDF documents with embedded tables, figures, and references. All PDFs are human-readable and can be downloaded individually.

The core of the study is the annotated master table of rules (in 03_rule_annotation/annotated_rules_master.pdf) and the final research paper (07_draft_paper/full_research_paper.pdf). The repository also includes guidelines, historical context, and supplementary materials.

### **How to use this repository**

    Start with 07_draft_paper/full_research_paper.pdf – it synthesises all findings.
    For detailed data, open 03_rule_annotation/annotated_rules_master.pdf.
    For methodology, see 03_rule_annotation/annotation_guidelines.pdf.
    For historical background, browse 04_historical_context/ PDFs.
    Figures and images are stored in 06_visualizations/; each PDF indicates where to insert them. You may generate images as described in the figure captions.

For each figure, I give a brief description and a suggested search query (if applicable). Example: "Figure 3.2: Map of the spread of Vinaya traditions. Search Wikimedia Commons for 'Buddhist monastic sites first millennium CE'."
